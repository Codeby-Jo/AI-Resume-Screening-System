# AI Resume Screening System

An AI-powered Resume Screening Web Application built using FastAPI, Python, Machine Learning, HTML, CSS, and JavaScript.

This project analyzes resumes based on a given Job Description and ranks candidates using AI similarity matching.

---

# Features

- Upload Multiple Resumes
- AI-Based Resume Ranking
- Job Description Matching
- Skill Extraction
- Candidate Status Detection
- Email Extraction
- Phone Number Extraction
- Experience Detection
- Resume Match Percentage
- Modern Frontend UI
- FastAPI Backend API
